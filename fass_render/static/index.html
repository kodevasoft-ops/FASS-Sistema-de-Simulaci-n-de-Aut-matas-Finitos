Construcción de Thompson — Regex → AFN-ε
Convierte una Expresión Regular en un AFN-ε (Autómata Finito No Determinista con transiciones épsilon) usando la construcción de Thompson (1968). Es el primer paso de la cadena clásica de un analizador léxico.
(a|b)*abb a*b+ (0|1)*00 a(b|c)*d ab?c
Operadores: a símbolo   ab concat   a|b unión   a* Kleene   a+ positiva   a? opcional   () grupo   ε épsilon

Minimización de Hopcroft — AFD → AFD Mínimo
Produce el AFD mínimo equivalente: el autómata con el menor número posible de estados que reconoce el mismo lenguaje. Usa el algoritmo de Hopcroft (1971) con complejidad O(n·|Σ|·log n). Permite verificar si dos AFD son equivalentes.
